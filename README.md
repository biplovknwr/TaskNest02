Overview

TaskNest is an Android task-management application prototype created as part of Assignment 2. The purpose of this version is to set up the basic structure of the app, implement initial screens, connect Firebase to the project, and show the early-stage UI as planned in Assignment 1.

This version focuses on building the foundation of the app so that it can be expanded and improved in Assignment 3.

Features Implemented in Assignment 2

Initial App Structure

Created the base Android Studio project for TaskNest.

Set up the package structure and main activity.

Added placeholder screens for login, registration, and task list, consistent with Assignment 1 wireframes.

Firebase Setup

Connected the project to Firebase.

Added google-services.json.

Integrated Firebase Authentication (email and password).

Integrated Firebase Firestore (initial setup only).

Successfully tested authentication connections.

Basic UI Implementation

Login Screen: email and password fields with a login button.

Registration Screen: email and password fields with a register button.

Task List Screen: shows placeholder items and basic layout.

Navigation between Login → Register → Task List.

Jetpack Compose Usage

Implemented screens using Jetpack Compose components.

Applied basic Material design elements.

Simple card-style layout placeholders for tasks.
